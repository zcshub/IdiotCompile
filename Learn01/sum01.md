## 编译程序 1 + 2 + 3 到栈式计算机

### 语法树构建
```cgo
          +
       +     3
     1   2
```


### 代码生成
后序遍历
push 1
push 2
add
push 3
add

执行：
push 1压栈
push 2压栈
add取出两个操作数[2,1]，得到的结果3压栈
push 3压栈
add取出两个操作数[3,3]，得到的结果6压栈
